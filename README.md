## ✨ المميزات

- 🎨 **تصميم داكن احترافي** - تصميم أسود أنيق مع تدرجات ملونة

## 📄 الصفحات

### 🏠 الصفحة الرئيسية (`/`)
- نبذة شخصية مع صورة
- عرض الخدمات الرئيسية
- قسم العملاء المميزين
- دعوة للتواصل

- السيرة الذاتية المفصلة
- الإحصائيات والإنجازات
- المهارات والخبرات
- الخبرة العملية

### 🎨 محفظة التصميم (`/portfolio-design`)
- عرض مشاريع التصميم
- الخدمات المقدمة
- أمثلة من الأعمال

### 📷 محفظة التصوير (`/portfolio-photography`)
- معرض أعمال التصوير
- أنواع التصوير المختلفة
- مشاريع مميزة

### 🎙️ محفظة التعليق الصوتي (`/portfolio-voice`)
- عينات صوتية
- أنواع التعليقات الصوتية
- المميزات والقدرات

### 🎓 الدورات التدريبية (`/courses`)
- دورة الإلقاء والخطابة
- دورة التعليق الصوتي
- ورش العمل المتخصصة

## 🎨 التخصيص

### تغيير الألوان
الألوان الرئيسية في `src/styles/theme.css`:
- الأزرق: `#3b82f6`
- البنفسجي: `#a855f7`
- الوردي: `#ec4899`
- الأخضر: `#10b981`

### تحديث المعلومات الشخصية

#### في `shared/Header.tsx`:
```tsx
<Link to="/" className="text-xl font-bold">
  مصطفى المحترف  {/* غير الاسم هنا */}
</Link>
```

#### في `shared/Footer.tsx`:
```tsx
<a href="mailto:info@example.com">  {/* غير البريد */}
  info@example.com
</a>
<a href="tel:+966123456789">  {/* غير رقم الهاتف */}
  +966 12 345 6789
</a>
```

### استبدال الصور
استبدل روابط الصور من Unsplash بصورك الشخصية في كل صفحة:
```tsx
<ImageWithFallback
  src="رابط_صورتك_هنا"
  alt="وصف الصورة"
  className="..."
/>
```
